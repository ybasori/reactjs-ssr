# ReactJS server-side renderer

Stack used:
- Typescript
- Sassy CSS
- ReactJS
- ExpressJS

Run app in development with yarn:
```shell
yarn start
```
or

with npm:
```shell
npm run start
```

Build app in production with yarn:
```shell
yarn build
```
or

with npm:
```shell
npm run build
```

then start in production:
```
node server/dist
```
